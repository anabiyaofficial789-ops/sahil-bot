## ⚡ Quick Deploy (Hostinger)

```bash

# 2. Terminal mein yeh commands chalayein:

npm install
cp .env.example .env
nano .env    # Sirf ADMIN_PASSWORD aur SESSION_SECRET badlein

# 3. PM2 se start karein:
npm install -g pm2
npm run pm2-start
pm2 save
pm2 startup
```

**Hostinger Settings:**
- Entry Point: `node web/server.js`
- Node.js Version: 20+
- Port: 8000
